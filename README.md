# ?? Student Performance Predictor

### ?? Live Demo — https://student-performance-predictor-01n6.onrender.com

An AI-powered web application that predicts student math scores using Machine Learning — built with Flask, trained on 1,000+ student records, and deployed live.

## ?? Features
- Math Score Prediction using Random Forest ML algorithm
- 7 Input Features — Gender, Race, Parental Education, Lunch, Test Prep, Reading & Writing Scores
- Performance Grade — A+, A, B+, B, C, D grading
- Radar Chart & Bar Chart — Visual score comparison
- Personalized Suggestions — Study tips based on predicted score
- Dark AI Theme — Professional modern UI

## ?? Quick Start
git clone https://github.com/kmishra2026/student-performance-predictor.git
pip install -r requirements.txt
python train.py
python app.py

## ?? ML Details
- Algorithm: Random Forest Regressor
- Dataset: Students Performance in Exams (Kaggle)
- Training Records: 1,000 students
- Input Features: 7
- Target: Math Score (0-100)

## ?? Deployment
Deployed on Render using Gunicorn
Live: https://student-performance-predictor-01n6.onrender.com
